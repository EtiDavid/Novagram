# Realtime Chat Application

A realtime chat application built for DevOps practice:
Docker • AWS ECS • CI/CD • Monitoring • Debugging Discipline

## Tech Stack

Frontend: React  
Backend: Node.js + Express + Socket.IO  
Database: MongoDB Atlas  
Deployment: AWS ECS Fargate + ALB  
CI/CD: GitHub Actions  

## Goals

- Learn Docker containerization properly
- Practice ECS deployments
- Implement real CI/CD pipelines
- Document process
