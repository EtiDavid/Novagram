# Novagram Web Chat Application

## Overview
Novagram is a full-stack web chat application built to simulate real-world production workflows, not just application features.


## Tech Stack
Frontend: React  
Backend: Node.js + Express + Socket.IO  
test: Jest + Supertest
Database: MongoDB Atlas  
Deployment: AWS ECS Fargate + ALB  
CI/CD: GitHub Actions  

## Project Status

Active development (ongoing). The project evolves as new tools and concepts are learned (e.g. Jenkins, Terraform, observability).

## Infrastructure

- Docker (multi-stage builds)
- Docker Compose (local orchestration)
- GitHub Actions (CI/CD)
- Docker Hub (image registry)
- AWS ECS Fargate (production deployment)
- AWS Secrets Manager (runtime secrets)
